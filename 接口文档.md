# 接口文档

##### 根地址
* http://www.goldlone.cn/sc

### 状态码列表

| 状态码 | 含义 |
| :---: | :--: |
| 1001  | 请求成功 |
| 2001  | 请求结果不存在 |
| 2002  | 操作失败 |
| 2003  | 权限不够 |
| 2004  | 登录失效 |
| 2005  | 请求参数缺失 |
| 3001  | 抛出异常 |

注：1001，3001状态码每个API都可能出现，一下说明文档不做赘述


### 请求返回消息格式

|字段名|类型|含义|
|:-|:-|:-|
|code|int|状态码|
|msg|string|返回信息|
|data|数组|数据(可能为空)|
